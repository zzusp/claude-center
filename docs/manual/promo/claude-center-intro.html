Web 控制台

任何设备访问：手机 / 平板 / PC

远程发布、监控、验收一站完成，不必守在跑任务的机器旁。

✓任务调度 · 草稿、定时、前置依赖、批量与状态流转
✓实时对话 · 指定 Worker + 项目分支，多轮远程协作
✓验收闭环 · PR 状态、合并检测、失败重试、取消在途
✓权限协作 · viewer / commenter / publisher / admin

控制面 · DB + SSE

PostgreSQL 唯一权威 + 可选实时线

任务、会话、Worker 心跳、权限与 Claude Code transcript 全部可追溯。

✓唯一权威 · tasks / workers / projects / sessions 全落库
✓SSE Relay · 项目 / 机器频道亚秒级推送
✓不可用退轮询 · 中转断线自动退回 DB 轮询，功能不降级
✓调度规则 · 多 Worker 认领队列，按并发上限运行

桌面 Worker

Windows / macOS 本地运行

连接本机 Claude Code 终端，认领队列并在独立 worktree 中执行。

✓并发执行 · max parallel + 队列认领
✓worktree 隔离 · 同仓多任务互不干扰
✓无人值守 · 自动回复、用量监控、远程控制
✓会话沉淀 · session.jsonl 实时同步回库
