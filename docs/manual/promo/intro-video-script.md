# 项目介绍视频 · 制作脚本（任务流转转场动画）

> **定位**：~20 秒端到端转场动画 · 拉新向 · 真实截图 + 自制终端动画 + 即梦光点转场。
>
> **整片灵魂**：一个光点 **= 一个任务**，镜头跟着它在四个端之间流动：
>
> ```
> 浏览器/手机 发任务 → 桌面 Worker 认领 → CLI 终端 执行 → git 远程仓库（PR 合并）
> ```

## ⚠️ 配色必须贴合项目「Claude Light」设计系统

全部取色来自 `apps/console/app/globals.css:5-46`，设计哲学（同文件 `:1-3`）是 **「信息 > 视觉，状态 > 装饰，结构 > 风格」**——所以视频要**亮色、克制、干净**，不是深色科技炫酷风。

| 角色 | 取值（CSS 变量）|
|---|---|
| 背景基调 | 暖白 `#f8f8f6`(--bg) + 极淡 stone 网格 `rgba(231,229,228,.5)` 64px |
| 面/卡 | `#ffffff` / `#fafaf9` / `#f5f5f4` |
| 边框 | `#e7e5e4`(--border) / `#d6d3d1`(--border-strong) |
| 文字 | `#1c1917` / `#44403c` / `#78716c` / `#a8a29e`（stone 暖灰）|
| **running 蓝** | `#2563eb` ← 「任务光点」主色（运行中）|
| **success 绿** | `#16a34a` ／ **merged 翠绿** `#059669` ← 光点终点（合并成功）|
| pending 琥珀 | `#f59e0b`　waiting 青 `#0891b2`　scheduled 紫 `#7c3aed` |
| 终端窗 | 底 `#1c1917`、字 `#fafaf9`、绿 `#86efac`、蓝 `#93c5fd` |
| 字体 | Inter（sans）/ JetBrains Mono（mono）|
| 圆角 / 阴影 | 8–12px / 极轻（hover only）|

**光点的状态色叙事**：光点全程是 running 蓝 `#2563eb`（= 一个运行中的任务），到终点 git 仓库时渐变为 merged 翠绿 `#059669` + 绿对勾——这正是产品里「时间线 active=蓝、done=绿」的语义（`globals.css:1144-1151`）。

## 即梦 3 段详细脚本

**风格总纲（每条 prompt 都已包含，三段必须一致）**：极简扁平的 **2D motion graphics / 高端 SaaS 界面微动效**风格；明亮暖白背景 + 极淡浅灰网格；一个蓝色发光圆点沿细引导线**平滑匀速**移动；镜头固定；干净、留白多、克制。
**反向约束（关键，压住即梦默认的炫酷倾向）**：不要深色满屏背景、不要霓虹/赛博朋克、不要大量粒子或光爆、不要镜头光晕、不要 3D 科技隧道、不要写实质感、不要任何文字/字母/Logo/人物。

### 转场 1 ｜ 浏览器/手机 → 桌面 Worker
```
极简扁平的 2D motion graphics 动效，高端 SaaS 界面微动效风格。明亮的暖白色背景（#f8f8f6），带极淡的浅灰色方格网格。一个发光的蓝色圆点（#2563eb，柔和光晕）代表一个任务，从画面左侧出现，沿着一条细的浅灰色（#d6d3d1）水平引导线，平滑匀速地向右移动，圆点后拖着一小段柔和的淡蓝色短轨迹。画面干净、留白多、现代、克制，镜头固定不动。明亮简洁。不要深色背景，不要霓虹或赛博朋克，不要大量粒子或镜头光晕，不要3D科技隧道，不要写实质感，不要任何文字、字母、Logo、人物。4K 高清。
```
**首尾**：左侧出现 → 右侧将出画（始终暖白）　**衔接**：端1 `tasks` 交叉淡化 → 端2 `worker`　**验收**：蓝点 #2563eb·匀速左→右·背景暖白·无炫酷粒子·无文字·4s

### 转场 2 ｜ 桌面 Worker → CLI 终端
```
极简扁平的 2D motion graphics 动效，高端 SaaS 界面微动效风格。明亮的暖白色背景（#f8f8f6）带极淡浅灰网格。一个蓝色发光圆点（#2563eb）从左侧沿浅灰色引导线平滑滑入画面中央一个深色（#1c1917）圆角终端窗口图形；进入瞬间，终端窗口内自上而下浮现几行柔和的绿色（#86efac）与蓝色（#93c5fd）等宽代码行的发光剪影，像代码在终端里运行。画面干净克制现代，明亮简洁，镜头固定。除中央终端图形是深色外，背景保持明亮暖白。不要深色满屏背景，不要霓虹赛博朋克，不要大量粒子，不要可辨认的真实文字，不要Logo、人物。4K 高清。
```
**首尾**：左侧滑入 → 代码在终端内浮现（背景仍暖白）　**衔接**：端2 `worker` 淡化 → **端3 `terminal.mp4`**（代码浮现正好引出打字执行）　**验收**：蓝点入终端·终端内是抽象代码剪影非真字·背景暖白·4s

### 转场 3 ｜ CLI 终端 → git 远程仓库
```
极简扁平的 2D motion graphics 动效，高端 SaaS 界面微动效风格。明亮的暖白色背景（#f8f8f6）带极淡浅灰网格。一个发光圆点从左侧深色（#1c1917）圆角终端窗口图形中飞出，沿着一条细的、先分叉再汇合的浅灰色 git 分支曲线平滑向右移动；移动过程中圆点颜色由蓝色（#2563eb）平滑渐变为翠绿色（#059669）；终点汇入右侧一个简洁的圆角仓库/数据库图标，图标随即亮起翠绿色（#059669）的对勾，表示合并成功。画面干净克制现代，明亮简洁，镜头固定。不要深色满屏背景，不要霓虹赛博朋克，不要大量粒子或光爆，不要任何文字、字母、Logo、人物。4K 高清。
```
**首尾**：左侧（终端）飞出 → 右侧仓库亮绿对勾　**衔接**：端3 `terminal.mp4` 淡化 → 端4 `pr-merged`（光点变绿呼应 PR 绿 Merged）　**验收**：蓝→绿渐变·终点绿对勾·背景暖白·无文字·4s

## 即梦操作 + 抽卡技巧
1. `AI 视频 → 文生视频 → 粘 prompt → 时长 4–5s / 选画幅 → 生成`。
2. **即梦默认偏科幻炫酷**，要靠 prompt 里「极简扁平 / 2D motion graphics / 明亮暖白背景」反复压；若仍跑偏深色科技风，prompt 开头再加「扁平矢量动画风格 flat vector animation」。
3. 抽卡挑「背景暖白 + 蓝点匀速 + 无粒子炫光 + 无文字」那条，下载 1080p+。
4. 三段共用同一套描述，保证风格统一。先各抽 1 条看整体连贯再补抽，省额度。

> **备选（更稳）**：这 3 段「光点沿线滑动 + 变色 + 打勾」本质就是项目克制风的 UI 动效，**我可以用 SVG+CSS 代码生成**（和 `terminal.mp4` 一个路子）——配色 100% 精确、可控、免费、不用抽卡。要的话说一声，我直接出 `transition-1/2/3.mp4`。

## 剪映合成时间轴（约 20 秒）
```
端1 tasks 截图    2.5s  缩放推近，蓝光点从「发布」处亮起 → 交叉淡化
转场1            4s    蓝光点飞向桌面
端2 worker 截图   2.5s  蓝光点落入、Worker 亮起 → 交叉淡化
转场2            4s    蓝光点钻进终端
端3 terminal.mp4  5~6s  整段用（自带打字节奏）→ 交叉淡化
转场3            4s    光点变绿飞向仓库，绿对勾亮起
端4 pr-merged 图  3s    绿色 Merged 打钩，轻微放大定格
```
- **统一光点**：全程一个蓝色小圆点（终点变绿）跟着飞，强化「同一个任务在流动」。
- **字幕**：4 个词跟着光点 → `你发任务` → `Worker 认领` → `终端执行` → `PR 自动合并`；字体用 Inter / 思源黑体，文字色 stone `#1c1917`。
- **BGM**：克制、明快、现代的轻电子（别用激烈科幻乐，跟亮色克制气质一致）；旁白可加可不加。
